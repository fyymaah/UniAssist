# 🎓 UniAssist - Web-Based Assistive Platform

## 📋 Description
UniAssist is a web-based assistive platform designed to address the limited accessibility of digital platforms for students with disabilities. It integrates multiple assistive technologies into a single, unified solution.

## ✨ Features
- **Dyslexia Support**: Text-to-Speech, Font Adjustments, Dyslexia-Friendly Fonts
- **Dysgraphia Support**: Speech-to-Text, Writing Aid
- **Color Identification**: Image Upload Color Detection, Color Palette Selector
- **Communication Board**: Keyword Speech, Text-to-Speech
- **Customization**: Dark Mode, Font Size, Accent Colors
